# Simple Todo List

A lightweight and easy-to-use desktop to-do list application built with PySide6.

---

## Features

- Add new tasks (via button or Enter key)
- Mark tasks as completed (checkbox)
- Delete selected tasks
- Clean and minimal UI

---

## Requirements

- Python 3.8+
- PySide6

---

## Installation

**1. Clone the repository:**
```bash
git clone https://github.com/your-username/simple-todo.git
cd simple-todo
```

**2. Install dependencies:**
```bash
pip install PySide6
```

**3. Run the application:**
```bash
python todo.py
```

---

## Usage

| Action | How to do it |
|---|---|
| Add a task | Type in the input box and click **Add Task** or press **Enter** |
| Mark as completed | Check the checkbox next to the task |
| Delete a task | Select the task and click **Delete Selected** |

Completed tasks are shown with a `(Completed)` label in the list. Unchecking the checkbox automatically removes the label.
